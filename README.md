##安装使用步骤

#1. npm install 
    下载需要的包

#2. npm start
    启动项目

#3. 浏览器访问 
    http://localhost:3000/snake

#4. 使用
    使用键盘的 上下左右箭头，或者  W, S, A, D 按钮，控制上下左右方向


###思路
1. 首先画出背景
2. 然后初始化初始蛇的位置，蛇为黑色
3. 蛇可以定时移动
4. 通过键盘控制蛇的移动
5. 显示食物为红色
6. 蛇吃完食物变长，继续行走，蛇头撞到自己失败，重新开始
7. 蛇的每个部分统一行走，转换方向时，后一个按照前一个的轨迹进行行走
8. 超过边界时，重新开始
9. 暂停，游戏暂停，蛇不动
10. 开始，蛇继续行走
11. 计分，每吃一个食物，则加一分
12. 重新开始，回到初始化状态
13. 优化界面，对页面布局进行优化
